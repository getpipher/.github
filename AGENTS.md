# getpipher/.github

> Organization-wide GitHub meta repo for the getpipher organization — the Pi coding-agent ecosystem (extensions, skills, tools built for and with Pi). Terminal-native, agent-first, open source.

## What lives here

- **`README.md`** — the organization profile README shown on the `github.com/getpipher` landing page (ecosystem overview + logo).
- **`assets/logo*.png`** — org logo, referenced by other getpipher repos' READMEs (e.g. `zai-media` hero).
- **`LICENSE`** — MIT, org-wide default.

## Notes

- This is GitHub's special `.github` meta repo: files here provide org-level default community-health files unless a repo has its own override.
- No GitLab mirror for getpipher repos.
- Keep `assets/logo.png` stable — downstream READMEs hotlink it via the `raw.githubusercontent.com/getpipher/.github/main/assets/logo.png` URL.
